============
Installation
============


Stable release
--------------

To install Image Drug VAEs, run this command in your terminal:

.. code-block:: console

    $ pip install image_drug_vaes

This is the preferred method to install Image Drug VAEs, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Image Drug VAEs can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/aclyde11/image_drug_vaes

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/aclyde11/image_drug_vaes/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/aclyde11/image_drug_vaes
.. _tarball: https://github.com/aclyde11/image_drug_vaes/tarball/master
